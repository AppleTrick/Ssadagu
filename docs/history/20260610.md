## 3. 계좌 인증 페이지 UX 개선 (VerifyAccountPage)

### 계좌번호 자동 입력
- 기존: 사용자가 직접 계좌번호를 입력
- 변경: 페이지 진입 시 `useMyAccount` 훅으로 본인 계좌 조회 → 자동 입력 후 `transfer-info` 단계로 바로 진입
- 안내 문구 추가: "회원가입 시 자동 생성된 한국은행 SSAFY 가상계좌로 인증이 진행됩니다."

### 1원 송금 후 입금 내역 카드 표시
- 1원 송금 완료 후 2초 대기 → 거래내역 API 조회 (`transactionType=A`, `orderByType=DESC`)
- 응답에서 `transactionSummary`가 `SSADAGU`로 시작하는 항목을 찾아 카드로 표시
- 카드 표시 항목: 입금액 / 입금자명 / 거래일시
- 로딩 중 shimmer 스켈레톤 애니메이션 표시
- 조회 실패 시 "잠시 후 입금 내역이 표시됩니다" 폴백 메시지

### VerificationFilter 수정
- `UNVERIFIED` 상태 사용자도 인증 진행 중 거래내역 조회가 가능하도록 화이트리스트에 추가
  ```
  /api/v1/demand-deposits/accounts/*/transactions
  ```

## 4. 위치 인증 페이지 Geolocation 에러 메시지 개선

**파일**: `useCurrentLocation.ts`

- 기존: 에러 종류에 상관없이 "위치 접근 권한이 거부되었습니다. 브라우저 설정을 확인해주세요." 고정 출력
- 변경: 에러 코드별 메시지 분기

| 에러 코드 | 메시지 |
|-----------|--------|
| `PERMISSION_DENIED` | 브라우저 주소창 자물쇠 아이콘에서 위치 허용 안내 |
| `POSITION_UNAVAILABLE` | OS 레벨 위치 서비스 문제 → 지도 직접 선택 안내 |
| `TIMEOUT` | 시간 초과 → 지도 직접 선택 안내 |

- Geolocation 옵션 변경
  - `enableHighAccuracy: false` (저전력, 빠른 응답)
  - `timeout: 15000` (10초 → 15초)
  - `maximumAge: 60000` (60초 캐시 허용, TIMEOUT 감소)
