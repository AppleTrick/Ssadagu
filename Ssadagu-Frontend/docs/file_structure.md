## 5. FSD л Ҳмқҙм–ҙ мқҳмЎҙм„ұ к·ңм№ҷ

```
app вҶ’ pages вҶ’ widgets вҶ’ features вҶ’ entities вҶ’ shared
```

- **мғҒмң„ л Ҳмқҙм–ҙлҠ” н•ҳмң„ л Ҳмқҙм–ҙл§Ң import к°ҖлҠҘ**
- к°ҷмқҖ л Ҳмқҙм–ҙ лӮҙ көҗм°Ё import кёҲм§Җ (мҳҲ: features к°„ м§Ғм ‘ import кёҲм§Җ)
- `shared`лҠ” лӘЁл“  л Ҳмқҙм–ҙм—җм„ң мӮ¬мҡ© к°ҖлҠҘн•ҳм§Җл§Ң, л№„мҰҲлӢҲмҠӨ лЎңм§Ғ нҸ¬н•Ё кёҲм§Җ
