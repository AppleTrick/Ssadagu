# 🎬 싸다구(Ssadagu) 서비스 시연 시나리오

본 문서는 평가 및 발표 시연을 위해 작성된 상세 시나리오입니다.
각 화면 이동 및 실행(클릭 위치, 입력값) 절차를 단계별로 설명합니다.

---

## 🟢 [단계 1] 회원가입 및 초기 설정 (Onboarding)

### 1-1. 회원가입 및 로그인 
- **화면**: 랜딩 페이지 / 로그인 화면
- **실행**: 
  - (신규 유저의 경우) `[회원가입]` 버튼 클릭 ➡️ 이메일, 닉네임, 비밀번호 입력 후 가입 완료.
  - (기존 유저의 경우) 이메일과 비밀번호 입력 후 `[로그인]` 클릭 ➡️ 홈 화면으로 이동.

### 1-2. 계좌 연동 및 1원 인증 (SSAFY 금융망)
- **화면**: 내 계좌 연동 화면 (`/verify-account`)
- **실행**:
  - 회원가입 후 계좌 연동 안내 팝업에서 `[내 계좌 연동하기]` 버튼 클릭.
  - 연동할 은행 선택 및 본인 계좌번호 입력 후 `[인증 요청]` 클릭.
  - SSAFY 금융망을 통해 계좌로 들어온 **1원 송금 내역(입금자명 인증번호)** 확인.
  - 화면 인증번호 입력칸에 해당 인증번호 입력 후 `[계좌 등록 완료]` 클릭.

### 1-3. 2차 결제 비밀번호 설정
- **화면**: 2차 비밀번호 설정 화면 (`/secondary-password-setup`)
- **실행**:
  - 안전한 송금/결제를 위한 6자리 핀(PIN) 번호 입력.
  - 확인을 위해 한 번 더 입력 후 `[설정 완료]` 클릭.

### 1-4. 동네(지역) 인증
- **화면**: 내 동네 설정 화면 (`/location-auth`)
- **실행**:
  - 브라우저/디바이스 GPS 권한 허용.
  - 현재 위치를 기반으로 내 동네가 텍스트로 표기됨 ➡️ `[동네 인증하기]` 클릭하여 지역 인증 완료.

---

## 🟡 [단계 2] 중고 상품 등록 및 탐색

### 2-1. 상품 등록
- **화면**: 홈 화면 하단 또는 우측 하단 `[+] (글쓰기)` 버튼 클릭 ➡️ 상품 등록 화면 (`/products/new`)
- **실행**:
  - 상품 이미지 업로드: `[사진 추가]` 영역 클릭 (테스트용 이미지 업로드).
  - 글 제목, 카테고리(예: 전자기기, 의류 등), 상품 설명 기재.
  - 희망 판매 가격 입력 (예: 50,000원).
  - 하단의 `[등록하기]` 버튼 클릭 ➡️ 등록 성공 및 상품 상세 페이지로 이동 확인.

### 2-2. 상품 탐색 및 상세 보기
- **화면**: 홈 메인 화면 (`/home`)
- **실행**:
  - 등록된 다양한 상품 리스트들을 스크롤하여 확인.
  - 시연자가 방금 등록한 상품 또는 다른 계정으로 등록해둔 상품 클릭.
  - 상품 상세 페이지에서 이미지 슬라이드, 판매자 정보, 가격, 상세 설명 확인.
  - 마음에 드는 상품 하트(찜하기) 아이콘 클릭하여 찜 활성화.

---

## 🔴 [단계 3] 채팅 및 실제 결제/거래 (핵심 플로우)

*이 단계는 판매자(A)와 구매자(B) 두 개의 브라우저/디바이스를 띄워두고 진행하면 가장 효과적입니다.*

### 3-1. 채팅방 입장 및 메시지 송수신
- **화면**: 상품 상세 페이지 ➡️ 채팅방 (`/chat`)
- **실행**:
  - 구매자(B)가 상품 상세 페이지 하단의 `[채팅하기]` 버튼 클릭.
  - 텍스트 입력창에 "안녕하세요, 직거래 가능할까요?" 입력 후 `[전송(종이비행기 아이콘)]` 클릭.
  - 판매자(A) 화면에서 알림 확인 또는 채팅 탭으로 이동하여 메시지 확인 및 답장.

### 3-2. 간편 결제 (송금) 요청 및 승인
- **화면**: 채팅방 내부 (`/chat`)
- **실행**:
  - **(결제 요청)** 구매자/판매자가 합의 후, 하단의 `[송금(또는 결제)]` 관련 버튼(예: `PAYMENT_REQUEST` 액션) 클릭.
  - 시스템 메시지로 채팅방 중앙에 `[결제 요청이 도착했습니다]` 카드가 렌더링됨.
  - **(결제 승인)** 구매자(B)가 해당 카드 내의 `[결제하기(송금하기)]` 버튼 클릭.

### 3-3. 2차 비밀번호 확인 및 거래 완료
- **화면**: 결제 모달 팝업
- **실행**:
  - 결제 금액(예: 50,000원)과 내 계좌 잔액 확인.
  - **1-3** 단계에서 설정했던 **[6자리 2차 비밀번호]** 키패드 입력.
  - 입력 완료와 동시에 SSAFY 금융망 API를 통한 실제 계좌 이체(출금/입금) 트랜잭션 진행.
  - 결제 성공 시 모달이 닫히며, 채팅방에 `[결제가 안전하게 완료되었습니다]` 시스템 메시지가 출력(`PAYMENT_SUCCESS`).

---

## 🔵 [단계 4] 마이페이지 및 내역 확인

### 4-1. 마이페이지 (내 정보 및 잔액)
- **화면**: 하단 네비게이션 `[마이/내정보]` 탭 (`/my`)
- **실행**:
  - 내 프로필(닉네임, 지역 등) 확인.
  - 연동된 계좌의 현재 **[남은 잔액]** 확인 (거래 이후 금액이 정상적으로 차감/증가했는지 실시간 확인).

### 4-2. 거래 내역 및 찜 목록 확인
- **화면**: 마이페이지 내부 메뉴
- **실행**:
  - `[나의 거래 내역]` 클릭 ➡️ 방금 수행된 거래가 '판매 완료/구매 완료' 혹은 '거래 성공' 상태로 리스트업 되어 있는지 확인.
  - `[관심 목록(찜)]` 클릭 ➡️ **2-2** 단계에서 찜했던 상품이 보이는지 확인 (취소 테스트를 위해 하트 해제 클릭 후 목록에서 사라지는 시연 추가 가능).
